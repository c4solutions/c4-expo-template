# C4 Expo Template

# to use as template

```
npx create-expo-app my-new-project --template https://github.com/c4solutions/c4-expo-template


```

## to run project

```
npx expo start
```

## production mode

```
npx expo start -p ios --no-dev --minify
```

## build development client

```
eas build --profile development --platform ios
```
